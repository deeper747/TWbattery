# 手動下載說明 — Taiwan Customs Data

## 資料來源

**財政部關務署 統計資料查詢平台**
URL: https://portal.sw.nat.gov.tw/APGA/GA30

---

## 每月更新步驟（每月 5 日後資料即可用）

### 步驟 1：開啟查詢頁面

前往 https://portal.sw.nat.gov.tw/APGA/GA30

### 步驟 2：填寫查詢條件（進口）

| 欄位 | 設定值 |
|------|--------|
| 進出口別 | **進口** |
| 資料週期 | **按月** |
| 起始年月 | `202001`（或你想要的起始月） |
| 結束年月 | 最新月份（例如 `202503`） |
| 貨品別 | **指定貨品號列**，輸入以下代碼（用逗號分隔）：|

```
850760,850790,283329,282200,282520,282540,283691,250410,380110,280300,382499,750210,281820,810520,390120,390210
```

| 欄位 | 設定值 |
|------|--------|
| 國家別 | **全部國家** |
| 統計值 | 進口值（千美元） |
| 輸出模式 | **下載 CSV** |

### 步驟 3：下載並儲存

將下載的 CSV 檔案存到 `data/raw/` 資料夾，命名為：

```
customs_I_YYYYMM_YYYYMM.csv
```

例如：`customs_I_202001_202503.csv`

### 步驟 4：重複以上步驟，改選「出口」

將出口 CSV 存為 `customs_E_202001_202503.csv`

### 步驟 5：更新 dashboard

```bash
python scripts/process_data.py
streamlit run dashboard.py
```

---

## CSV 預期欄位格式

從關務署下載的 CSV 通常包含以下欄位：

| 欄位 | 說明 |
|------|------|
| 年份 / 月份 | 資料年月 |
| 貨品號列 | 6 碼 HS Code |
| 貨品名稱 | 中文品名 |
| 國別 | 貿易夥伴國名 |
| 進（出）口值（千美元） | 以千美元計算的貿易值 |
| 淨重（公噸） | 重量（可能為空） |

> 如果欄位名稱與此不同，請修改 `scripts/process_data.py` 中的 `COLUMN_MAP` 對照表。
